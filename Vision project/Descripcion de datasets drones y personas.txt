Autonomous Drone Landing & Object Detection Dataset
Origin & Acknowledgments
This dataset is derived from the TEKNOFEST AI in Aviation Competition (Havacılıkta Yapay Zeka Yarışması).

Original Repository: TEKNOFEST GitHub Repository
This dataset provides a pre-processed version of the competition data (converted from Pascal VOC to YOLO format, structured into Train/Val splits, and cleaned) to facilitate immediate model training.

Context
This dataset is designed to aid in the development of computer vision systems for autonomous Unmanned Aerial Vehicles (UAVs), specifically for precision landing and obstacle avoidance tasks. It contains high-resolution aerial imagery captured from a drone, featuring specific landing markers (pads), human subjects, and vehicles.

The dataset is ideal for training models like YOLO, SSD, or Faster R-CNN to identify safe landing zones and avoid dynamic obstacles in real-time.

Dataset Content
The dataset consists of video frames extracted and annotated in Pascal VOC (XML) and converted to YOLO (TXT) formats.

Resolution: 1920x1080 pixels
Viewpoint: Top-down and angled aerial view (Drone footage)
Environment: Outdoor track and field complex
Annotation Formats:
.xml (Pascal VOC standard)
.txt (YOLO standard - Normalized coordinates)
Class Descriptions (Labels)
The dataset includes 4 distinct classes. Special attention has been given to distinguishing between different types of landing markers based on color and pattern.

Class ID	Class Name	Visual Description	Function / Purpose
0	Vehicle	Standard vehicles (cars) visible in the background.	Distractor objects / Obstacles.
1	UAP	Cyan/Blue circular pad with "UAP" text in the center.	Primary Landing Target. Represents the designated safe landing zone.
2	UAI	Red circular pad with a White Crescent symbol.	Secondary Landing Target. Represents a specialized mission zone or flag area.
3	Person	Human subjects moving on the field.	Safety detection (Do not land if a person is detected).
Use Cases
Researchers and developers can use this dataset to build:

Autonomous Landing Systems: Detecting the "UAP" or "UAI" markers to align the drone for descent.
Safety Mechanisms: Preventing landing when a Person or Vehicle is too close to the target.
Visual Servoing: Tracking the landing pad during flight to adjust velocity and position.

Aerial Semantic Drone Dataset
The Semantic Drone Dataset focuses on semantic understanding of urban scenes for increasing the safety of autonomous drone flight and landing procedures. The imagery depicts more than 20 houses from nadir (bird's eye) view acquired at an altitude of 5 to 30 meters above the ground. A high-resolution camera was used to acquire images at a size of 6000x4000px (24Mpx). The training set contains 400 publicly available images and the test set is made up of 200 private images.

This dataset is taken from https://www.kaggle.com/awsaf49/semantic-drone-dataset. We remove and add files and information that we needed for our research purpose. We create our tiff files with a resolution of 1200x800 pixel in 24 channel with each channel represent classes that have been preprocessed from png files label. We reduce the resolution and compress the tif files with tiffile python library.

If you have any problem with tif dataset that we have been modified you can contact nunenuh@gmail.com and gaungalif@gmail.com.

This dataset was a copy from the original dataset (link below), we provide and add some improvement in the semantic data and classes. There are the availability of semantic data in png and tiff format with a smaller size as needed.
